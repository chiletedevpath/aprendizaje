# Patrones de Diseno

Esta carpeta contiene practica de principios SOLID y patrones de diseno, organizada por concepto para diferenciar teoria aplicada, ejercicios de clase y ejemplos con Spring Boot.

## Estructura

- `principios-solid`: ejercicios sobre SRP, OCP, LSP, ISP, DIP y practicas calificadas.
- `creacionales`: patrones creacionales como Builder, Factory, Prototype y Singleton.
- `estructurales`: patrones estructurales como Adapter y Facade.
- `practica-integrada`: ejercicios donde se combinan principios SOLID y patrones en un mismo caso.
- `evaluaciones`: evaluaciones, PC o practicas calificadas relacionadas con patrones.

## Convencion

- `java`: ejercicios de clase o laboratorio.
- `spring-boot`: ejemplos implementados como proyectos Spring Boot.

## Estado

Contenido de aprendizaje en saneamiento progresivo para publicacion segura.
