# Principios SOLID

Ejercicios sobre principios de diseno orientado a objetos.

## Contenido

- `b_ocp`: principio abierto/cerrado.
- `c_lsp`: principio de sustitucion de Liskov.

## Proposito

Practicar decisiones de diseno que permitan extender comportamiento sin romper codigo existente y usar herencia o polimorfismo sin alterar contratos esperados.

## Uso comunitario

Puede convertirse en material de comunidad si cada ejercicio muestra una version problematica, una version corregida y una explicacion breve del principio aplicado.
