# Evaluaciones

Esta carpeta conserva practicas calificadas y ejercicios de evaluacion relacionados con patrones de diseno.

## Criterio de uso

El contenido se mantiene como evidencia academica dentro del repositorio de aprendizaje.

No debe publicarse en la futura web de Chilete DevPath como banco directo de preguntas, respuestas o evaluaciones. Si algun ejercicio aporta valor comunitario, debe adaptarse como caso propio, con redaccion nueva y sin conservar el formato de evaluacion.

## Contenido

- `pc2-design-patterns`: ejercicios asociados a practica o evaluacion de patrones.
- `practica_calificada`: ejercicios de practica calificada sobre principios y patrones.

## Uso comunitario

Solo se recomienda usarlo como insumo interno para crear guias originales de Chilete DevPath.
