# Practica integrada

Ejercicios donde se combinan principios SOLID y patrones de diseno en un mismo caso.

## Proposito

Practicar la seleccion de varias tecnicas de diseno dentro de una solucion completa, evitando tratar cada patron como una receta aislada.

## Criterio

Este contenido puede ser util para futuras guias, pero debe revisarse caso por caso antes de publicarse en la web. La prioridad es explicar que problema se resuelve y por que se eligio cada principio o patron.
