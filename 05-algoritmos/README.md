# Algorithms

[Spanish version](README.es.md)

This folder contains exercises, notes, and examples related to algorithms and problem solving.

## Topics

- `searching`: linear search, binary search, and related techniques.
- `sorting`: sorting algorithms and tradeoffs.
- `recursion`: base cases and recursive calls.
- `complexity-analysis`: time complexity, space complexity, and Big O.
- `problem-solving-strategies`: approaches for algorithmic problems.
