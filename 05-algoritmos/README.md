# Algoritmos

Ejercicios sobre análisis, diseño y aplicación de algoritmos.

## Temas

- `complejidad`: análisis de eficiencia.
- `busqueda`: búsqueda lineal, binaria y variantes.
- `ordenamiento`: algoritmos de ordenamiento.
- `recursion`: resolución mediante llamadas recursivas.
- `resolucion-problemas`: estrategias generales para resolver ejercicios.
