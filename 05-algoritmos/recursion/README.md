# Recursion

Ejercicios sobre funciones que se llaman a si mismas y problemas que se resuelven por exploracion.

## Contenido

- Recursion basica.
- Conversion a binario.
- Potencia y suma recursiva.
- Backtracking.
- Laberinto.
- Ocho reinas.
- Recorrido del caballo.

## Proposito

Comprender la diferencia entre caso base, caso recursivo y retroceso. Esta seccion ayuda a pensar problemas donde una decision abre nuevas rutas de solucion.

## Uso comunitario

Para publicarlo como material de comunidad, conviene acompanar cada ejercicio con diagrama de llamadas o tablero de ejemplo.
