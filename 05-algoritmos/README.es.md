# Algoritmos

[English version](README.md)

Esta carpeta contiene ejercicios, apuntes y ejemplos relacionados con algoritmos y resolucion de problemas.

## Temas

- `searching`: busqueda lineal, busqueda binaria y tecnicas relacionadas.
- `sorting`: algoritmos de ordenamiento y diferencias.
- `recursion`: casos base y llamadas recursivas.
- `complexity-analysis`: complejidad temporal, espacial y Big O.
- `problem-solving-strategies`: enfoques para problemas algoritmicos.
