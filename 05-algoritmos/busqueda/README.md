# Busqueda

Ejercicios sobre localizacion de elementos dentro de arreglos.

## Contenido

- Busqueda lineal.
- Busqueda binaria.
- Ordenamiento por insercion como paso previo para busqueda binaria.
- Comparacion entre buscar en datos desordenados y ordenados.

## Proposito

Entender que la busqueda lineal revisa elemento por elemento, mientras que la busqueda binaria reduce el rango de busqueda cuando los datos ya estan ordenados.

## Uso comunitario

Puede convertirse en una guia inicial si se agrega una pregunta clara: que cambia cuando el arreglo esta ordenado y por que la busqueda binaria necesita esa condicion.
