# Resolucion de problemas

Ejercicios sobre estrategias para analizar, organizar y buscar informacion.

## Contenido

- Introduccion a algoritmos.
- Busqueda secuencial ordenada y desordenada.
- Busqueda por bloques.
- Busqueda con indices.
- Archivos indexados y listas auxiliares.

## Proposito

Aplicar algoritmos a situaciones mas cercanas a problemas reales: buscar registros, organizar claves y comparar estrategias de acceso a informacion.

## Uso comunitario

Puede alimentar la futura web si se separan los ejercicios en guias pequenas con problema, analisis, solucion y prueba de escritorio.
