# Ordenamiento

Ejercicios sobre tecnicas para reorganizar datos.

## Contenido

- Mezcla directa.
- Fusion natural.
- Versiones basicas y optimizadas.
- Shell Sort.
- Serie Fibonacci como practica recursiva relacionada.

## Proposito

Practicar algoritmos que ordenan colecciones y observar como cambia el arreglo despues de aplicar una estrategia.

## Uso comunitario

Antes de llevar este contenido a la web, conviene agregar una tabla comparativa con idea general, ventaja, limite y caso de uso de cada algoritmo.
