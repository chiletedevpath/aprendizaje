# Ejercicios para la comunidad

Estos retos complementan los ejercicios existentes de `05-algoritmos`.

## Busqueda

1. Buscar un tema dentro de una lista usando busqueda lineal.
2. Ordenar una lista de puntajes y aplicar busqueda binaria.
3. Comparar cuantas revisiones hace cada estrategia.

## Ordenamiento

1. Ordenar notas de menor a mayor.
2. Ordenar recursos por nivel: basico, intermedio y avanzado.
3. Explicar con una tabla el estado de la lista despues de cada pasada.

## Recursion

1. Calcular la suma de los primeros `n` numeros.
2. Convertir un numero decimal a binario.
3. Resolver una potencia usando llamadas recursivas.

## Backtracking

1. Resolver un laberinto pequeno usando una matriz.
2. Explicar que caminos se descartan y por que.
3. Proponer una variante del problema de ocho reinas con tablero reducido.

## Criterio para resolver

Antes de programar:

1. Define entrada y salida.
2. Elige una estrategia.
3. Escribe un caso normal y uno limite.
4. Implementa una version pequena.
5. Explica el resultado y el costo practico de la solucion.
