# Glosario de programacion basica

Este glosario resume terminos que aparecen en los ejercicios de `02-programacion-basica`.

| Termino | Significado practico |
|---|---|
| Variable | Espacio con nombre para guardar un dato durante la ejecucion del programa. |
| Constante | Valor que no debe cambiar despues de ser definido. |
| Tipo de dato | Categoria que indica que clase de valor se guarda: entero, decimal, texto, booleano, entre otros. |
| Conversion implicita | Conversion que Java permite automaticamente cuando no hay perdida relevante de informacion. |
| Conversion explicita | Conversion forzada con casting. Puede perder informacion si el tipo destino es menor. |
| Operador | Simbolo que realiza una operacion, como suma, comparacion o asignacion. |
| Condicional | Estructura que permite ejecutar codigo segun una condicion. |
| Bucle | Estructura que repite instrucciones mientras se cumpla una condicion o por una cantidad definida. |
| Metodo | Bloque de codigo reutilizable que puede recibir parametros y devolver un resultado. |
| Parametro | Dato que recibe un metodo para trabajar. |
| Retorno | Valor que un metodo entrega como resultado. |
| Cadena | Texto representado con `String`. |
| Arreglo | Estructura que guarda varios valores del mismo tipo en posiciones numeradas. |
| Matriz | Arreglo bidimensional organizado por filas y columnas. |
| Caso de prueba | Entrada usada para comprobar si una solucion funciona como se espera. |
| Caso limite | Situacion extrema o especial que puede revelar errores. |
| Validacion | Revision de un dato antes de usarlo para evitar resultados incorrectos. |

## Criterio de uso

Cada termino debe entenderse desde la practica. Si una palabra no se puede explicar con un ejercicio propio, todavia necesita refuerzo.
