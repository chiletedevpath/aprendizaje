# Cadenas

Area de practica para manejo de texto, validacion de cadenas, reportes formateados y uso de `StringBuilder` en Java.

## Contenido

- `java/cadenas-basicas`: operaciones basicas con cadenas y validaciones.
- `java/string-builder`: ejemplos usando `StringBuilder` y operaciones `append`.
- `java/reportes-formateados`: ejercicios enfocados en reportes de texto formateado.
