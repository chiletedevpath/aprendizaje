# Basic Java Exercises: Arrays

[Spanish version](README.es.md)

Java practice exercises focused on arrays and array traversal.

## Exercises

- `DetectarNumerosDuplicados.java`: detects repeated numbers using nested loops.
- `AnalizarPromedioVentas.java`: calculates total sales, average sales, and values above or below the average.
- `AnalizarNotasAlumnos.java`: processes student names and grades using parallel arrays.

## Main Concepts

- Arrays
- Parallel arrays
- Traversal
- Counters
- Numeric processing
- Duplicate detection
