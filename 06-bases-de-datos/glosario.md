# Glosario de bases de datos

Este glosario resume conceptos usados en los ejercicios de `06-bases-de-datos`.

| Termino | Significado practico |
|---|---|
| Base de datos | Conjunto organizado de informacion. |
| Tabla | Estructura que guarda registros con columnas definidas. |
| Registro | Fila de una tabla. |
| Campo | Columna o atributo dentro de una tabla. |
| Entidad | Concepto del negocio que necesita almacenarse. |
| Relacion | Vinculo entre entidades. |
| Clave primaria | Campo que identifica de forma unica un registro. |
| Clave foranea | Campo que relaciona una tabla con otra. |
| DDL | Instrucciones para crear o modificar estructuras. |
| DML | Instrucciones para insertar, actualizar o eliminar datos. |
| Consulta | Instruccion para recuperar informacion. |
| Normalizacion | Organizacion de datos para reducir duplicidad e inconsistencias. |
| Diagrama ER | Representacion visual de entidades y relaciones. |
| Script SQL | Archivo con instrucciones que puede ejecutar un motor de base de datos. |

## Criterio de uso

Un modelo esta bien trabajado cuando permite responder consultas reales sin duplicar informacion innecesaria.
