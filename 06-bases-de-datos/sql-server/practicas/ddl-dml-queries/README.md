# SQL Server Practice: DDL, DML, and Queries

[Spanish version](README.es.md)

Class exercises for practicing structure creation, data manipulation, and queries in SQL Server.

## Contents

- `01-ddl-basico.sql`: database, table, and constraint creation and changes.
- `02-dml-basico.sql`: inserts, updates, deletes, and basic queries.
- `03-consultas-avanzadas.sql`: advanced select queries.
- `04-participacion-clase.sql`: short class participation exercise.
