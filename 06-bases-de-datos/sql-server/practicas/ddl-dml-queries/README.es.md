# Practica SQL Server: DDL, DML y Consultas

[English version](README.md)

Ejercicios de clase para practicar creacion de estructuras, manipulacion de datos y consultas en SQL Server.

## Contenido

- `01-ddl-basico.sql`: creacion y modificacion de base de datos, tablas y restricciones.
- `02-dml-basico.sql`: insercion, actualizacion, eliminacion y consultas base.
- `03-consultas-avanzadas.sql`: consultas de seleccion avanzada.
- `04-participacion-clase.sql`: ejercicio corto de participacion en clase.
