# Estructuras de Datos

[English version](README.md)

Esta carpeta contiene ejercicios, apuntes e implementaciones de estructuras de datos comunes.

## Temas

- `arreglos`: colecciones indexadas y operaciones basicas.
- `linked-lists`: estructuras lineales basadas en nodos.
- `stacks`: operaciones LIFO.
- `queues`: operaciones FIFO.
- `trees`: estructuras de datos jerarquicas.
- `graphs`: nodos, aristas y recorridos.
- `hash-tables`: acceso clave-valor.
