# Arreglos

Ejercicios introductorios sobre listas indexadas y operaciones basicas de colecciones.

## Contenido

- Creacion de listas.
- Insercion de elementos.
- Actualizacion con `set`.
- Eliminacion con `remove`.
- Revision del orden de los datos despues de cada operacion.

## Proposito

Practicar como una coleccion cambia cuando se agregan, reemplazan o eliminan elementos. Este bloque sirve como base antes de estudiar listas enlazadas, tablas hash y otras estructuras.

## Uso comunitario

Puede publicarse como ejercicio inicial si se agrega un enunciado propio y una breve explicacion del resultado esperado.
