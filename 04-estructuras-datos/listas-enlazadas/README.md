# Listas enlazadas

Ejercicios sobre colecciones lineales usando `ArrayList`, `LinkedList` y objetos simples.

## Contenido

- Creacion de colecciones de objetos.
- Recorrido con indice y con `for-each`.
- Insercion en una posicion especifica.
- Comparacion practica entre listas basadas en arreglo y listas enlazadas.

## Proposito

Relacionar la teoria de listas con implementaciones disponibles en Java. El objetivo no es memorizar una clase, sino entender que cada estructura tiene ventajas segun el tipo de operacion.

## Uso comunitario

Antes de llevarlo a la web, conviene agregar preguntas guia como: cuando conviene usar `ArrayList`, cuando conviene usar `LinkedList` y que costo tiene insertar al inicio de una lista.
