# Grafos

Ejercicios sobre representacion de grafos, recorridos y caminos.

## Contenido

- Grafo dirigido con BFS y DFS.
- Grafo no dirigido con BFS y DFS.
- Algoritmo de Dijkstra para caminos minimos.
- Algoritmo de Warshall para caminos alcanzables.

## Proposito

Practicar estructuras que representan relaciones entre elementos. Los grafos ayudan a modelar rutas, dependencias, conexiones y decisiones basadas en caminos.

## Uso comunitario

Para publicarlo en la web, conviene agregar diagramas o matrices de ejemplo que permitan comparar el grafo visual con la salida del programa.
