# Tablas hash

Ejercicios sobre dispersion, acceso por clave y manejo de colisiones.

## Contenido

- Tabla hash con encadenamiento mediante nodos.
- Busqueda, insercion y eliminacion de claves.
- Funciones de dispersion: modular, plegamiento y multiplicacion.
- Doble hashing para resolver colisiones.

## Proposito

Comprender como una clave puede transformarse en un indice y que ocurre cuando dos claves llegan a la misma posicion. Este tema conecta estructuras de datos con rendimiento de busqueda.

## Uso comunitario

El contenido puede convertirse en material explicativo si se acompana de una comparacion visual entre colision por encadenamiento y colision resuelta por doble hashing.
