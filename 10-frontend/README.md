# Frontend

Espacio para prácticas enfocadas en frontend moderno.

## Enfoque

Este módulo se diferencia de `07-desarrollo-web` porque se centra en construcción de interfaces con herramientas modernas, componentes reutilizables, manejo de estado, consumo de APIs y experiencia de usuario.

Las bases de HTML, CSS, JavaScript y PHP se organizan en `07-desarrollo-web`.

## Contenido

- `react`: ejercicios relacionados con componentes, propiedades, estado y composición de interfaces.
