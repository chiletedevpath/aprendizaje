# Glosario de desarrollo web

Este glosario resume conceptos usados en los ejercicios de `07-desarrollo-web`.

| Termino | Significado practico |
|---|---|
| HTML | Lenguaje para estructurar contenido web. |
| Etiqueta | Elemento que define una parte del documento. |
| Semantica | Uso de etiquetas que describen el significado del contenido. |
| CSS | Lenguaje para definir presentacion visual. |
| Selector | Regla que indica a que elementos se aplican estilos. |
| Flexbox | Modelo de CSS para distribuir elementos en una direccion. |
| Responsive | Capacidad de adaptarse a distintos tamanos de pantalla. |
| JavaScript | Lenguaje usado para interaccion y logica del lado del cliente. |
| Evento | Accion que ocurre en la interfaz, como clic o envio de formulario. |
| Asincronia | Trabajo que no bloquea la ejecucion principal. |
| PHP | Lenguaje usado en servidor para procesar solicitudes web. |
| Formulario | Grupo de campos que envia datos al servidor o a un script. |
| Metodo GET | Envio de datos visibles en la URL. |
| Metodo POST | Envio de datos dentro del cuerpo de la solicitud. |
| CRUD | Operaciones de crear, leer, actualizar y eliminar datos. |

## Criterio de uso

Una practica web esta bien construida cuando el contenido se entiende sin estilos, los estilos mejoran la experiencia y la interaccion no oculta la logica.
