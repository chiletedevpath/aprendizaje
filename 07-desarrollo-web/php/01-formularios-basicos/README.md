# Formularios Básicos con PHP

Esta carpeta contiene ejercicios iniciales donde se relacionan páginas HTML con scripts PHP.

## Objetivo

Comprender cómo un formulario web puede enviar datos para ser procesados por PHP.

## Archivos

| Archivo | Propósito |
|---|---|
| `index.html` | Página de entrada o navegación de los ejercicios. |
| `estilos.css` | Estilos compartidos para las páginas. |
| `ejercicio_1.html` / `ejercicio_1.php` | Primer ejercicio de formulario y procesamiento. |
| `ejercicio_2.html` / `ejercicio_2.php` | Segundo ejercicio de formulario y procesamiento. |
| `ejercicio_3.html` / `ejercicio_3.php` | Tercer ejercicio de formulario y procesamiento. |
| `ejercicio_4_get.html` / `ejercicio_4_get.php` | Ejercicio usando método `GET`. |
| `ejercicio_4_post.html` / `ejercicio_4_post.php` | Ejercicio usando método `POST`. |

## Temas practicados

- Formularios HTML.
- Envío de datos hacia PHP.
- Diferencia básica entre `GET` y `POST`.
- Separación entre estructura HTML, estilos CSS y procesamiento PHP.

## Recomendaciones

- Ejecuta los archivos desde un servidor local con soporte para PHP.
- Revisa primero el formulario HTML.
- Luego revisa el archivo PHP que recibe los datos.
- Prueba entradas normales y entradas vacías.
- Documenta errores o comportamientos inesperados.
