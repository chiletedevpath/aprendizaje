# Formularios basicos con PHP

Esta carpeta contiene ejercicios iniciales donde se relacionan paginas HTML con scripts PHP.

## Objetivo

Comprender como un formulario web puede enviar datos para ser procesados por PHP.

## Archivos

| Archivo | Proposito |
|---|---|
| `index.html` | Pagina de entrada o navegacion de los ejercicios. |
| `estilos.css` | Estilos compartidos para las paginas. |
| `ejercicio_1.html` / `ejercicio_1.php` | Primer ejercicio de formulario y procesamiento. |
| `ejercicio_2.html` / `ejercicio_2.php` | Segundo ejercicio de formulario y procesamiento. |
| `ejercicio_3.html` / `ejercicio_3.php` | Tercer ejercicio de formulario y procesamiento. |
| `ejercicio_4_get.html` / `ejercicio_4_get.php` | Ejercicio usando metodo `GET`. |
| `ejercicio_4_post.html` / `ejercicio_4_post.php` | Ejercicio usando metodo `POST`. |

## Temas practicados

- Formularios HTML.
- Envio de datos hacia PHP.
- Diferencia basica entre `GET` y `POST`.
- Separacion entre estructura HTML, estilos CSS y procesamiento PHP.

## Recomendaciones

- Ejecuta los archivos desde un servidor local con soporte para PHP.
- Revisa primero el formulario HTML.
- Luego revisa el archivo PHP que recibe los datos.
- Prueba entradas normales y entradas vacias.
