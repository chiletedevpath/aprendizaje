# PHP

Este módulo reúne prácticas iniciales de PHP dentro de la ruta de desarrollo web de Chilete DevPath.

El contenido actual corresponde a ejercicios del curso **Taller de Programación Web** de la carrera de **Ingeniería de Sistemas**. Se organiza como aprendizaje progresivo, no como proyecto académico final.

## Objetivo

Practicar el uso de PHP en páginas web, especialmente la comunicación entre formularios HTML y scripts PHP.

## Contenido

- `01-formularios-basicos/`: ejercicios iniciales con HTML, PHP, estilos y envío de datos.
- `glosario.md`: términos básicos del módulo.

## Ruta sugerida

1. Revisa el glosario.
2. Abre la carpeta `01-formularios-basicos/`.
3. Identifica qué archivos son formularios HTML y qué archivos procesan datos con PHP.
4. Prueba los ejercicios en un entorno local compatible con PHP.
5. Documenta qué aprendiste y qué errores encontraste.

## Criterio de organización

- Ejercicios y prácticas pequeñas de PHP viven en `aprendizaje/07-desarrollo-web/php/`.
- Proyectos finales o entregas académicas del curso deben referenciarse desde `academia`.
- Si más adelante se trabaja backend PHP avanzado, se evaluará si corresponde conectarlo también con `09-backend`.
