# PHP

Este modulo reune practicas iniciales de PHP dentro de la ruta de desarrollo web de Chilete DevPath.

El contenido actual corresponde a ejercicios del curso **Taller de Programacion Web** de la carrera de **Ingenieria de Sistemas**. Se organiza como aprendizaje progresivo, no como proyecto academico final.

## Objetivo

Practicar el uso de PHP en paginas web, especialmente la comunicacion entre formularios HTML y scripts PHP.

## Contenido

- `01-formularios-basicos/`: ejercicios iniciales con HTML, PHP, estilos y envio de datos.
- `02-ciclos-y-arreglos/`: ejercicios de ciclos, calculos repetitivos y listas simples en PHP.
- `glosario.md`: terminos basicos del modulo.

## Ruta sugerida

1. Revisar el glosario.
2. Abrir `01-formularios-basicos/`.
3. Identificar que archivos son formularios HTML y que archivos procesan datos con PHP.
4. Continuar con `02-ciclos-y-arreglos/` para practicar repeticion y procesamiento de listas.
5. Probar los ejercicios en un entorno local compatible con PHP.

## Criterio de organizacion

- Ejercicios y practicas pequenas de PHP viven en `aprendizaje/07-desarrollo-web/php/`.
- Proyectos finales o entregas academicas del curso deben referenciarse desde `academia`.
- Si mas adelante se trabaja backend PHP avanzado, se evaluara si corresponde conectarlo tambien con `09-backend`.
