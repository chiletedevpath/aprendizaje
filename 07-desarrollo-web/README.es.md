# Frontend

[English version](README.md)

Esta carpeta contiene ejercicios, apuntes y ejemplos relacionados con desarrollo frontend.

## Temas

- `html`: estructura semantica y marcado de documentos.
- `css`: estilos, layout y diseno responsive.
- `javascript`: logica del navegador, DOM y fundamentos del lenguaje.
- `react`: componentes, props, estado, hooks y composicion de interfaces.
- `ui-practice`: pequenos ejercicios de interfaz.
