# HTML

Ejercicios sobre estructura de documentos web.

## Contenido

- Uso de etiquetas base.
- Jerarquia de titulos y secciones.
- Enlaces y contenido textual.
- Tablas y listas como practica academica inicial.
- Plantilla de CV web con datos ficticios.

## Proposito

Practicar la estructura de una pagina antes de incorporar estilos, interaccion o frameworks.

## Uso comunitario

Puede publicarse como material introductorio si se acompana con explicaciones de estructura semantica y buenas practicas actuales.
