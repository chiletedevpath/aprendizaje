Propiedades Flexbox

flex-grow

Permite que el elemento crezca y ocupe mas espacio disponible.

order

Modifica el orden visual de los elementos flexibles.

align-self

Permite una alineacion individual dentro del contenedor flexible.

flex-basis

Define el tamano inicial base de un elemento flexible.
