Propiedades Flexbox

flex-grow

Permite que el elemento crezca y ocupe más espacio disponible.

order

Modifica el orden visual de los elementos flexibles.

align-self

Permite una alineación individual dentro del contenedor flexible.

flex-basis

Define el tamaño inicial base de un elemento flexible.
