# CSS

Ejercicios sobre estilos, layout y presentacion visual de paginas web.

## Contenido

- Enlace entre HTML y CSS.
- Selectores basicos.
- Propiedades visuales.
- Practica de CSS3.
- Maquetacion inicial.

## Proposito

Entender como CSS transforma la estructura HTML en una interfaz visual organizada.

## Uso comunitario

Antes de llevarlo a la web, conviene agregar capturas o comparaciones entre la estructura HTML sin estilos y la version estilizada.
