# JavaScript

Ejercicios iniciales de JavaScript dentro de la ruta de desarrollo web.

## Contenido

| Carpeta | Enfoque |
|---|---|
| `asincronia/cafeteria-asincrona` | Promesas, `async/await`, `setTimeout` y manejo de errores. |

## Proposito

Practicar interaccion y logica del lado del cliente antes de pasar a frameworks o proyectos frontend mas avanzados.

## Criterio Chilete DevPath

Los ejercicios pueden convertirse en guias para comunidad si se documentan con problema, solucion paso a paso y resultado esperado.
