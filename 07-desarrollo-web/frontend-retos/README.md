# Frontend Retos

Practicas pequenas de frontend realizadas durante la formacion complementaria en Tecsup.

## Autor

- Chilete DevPath
- Estudiante de Ingenieria de Sistemas e Informatica

## Contexto formativo

- Institucion: Tecsup
- Curso: Fullstack con Java
- Enfoque: Frontend
- Tipo: Practicas de aprendizaje

## Por que se hizo

Esta carpeta conserva practicas pequenas de frontend sin convertir cada ejercicio en un proyecto independiente.

La idea es agrupar retos breves que refuerzan HTML, CSS, JavaScript y programacion asincrona.

## Reto 1: Urban Food

Practica visual de menu web.

Tecnologias:

- HTML
- CSS
- Imagenes locales

Contenido detectado:

- Pagina `index.html`.
- Estilos en `styles.css`.
- Tarjetas de productos.
- Imagenes de cafe, hamburguesa y pizza.

Nota: las imagenes locales se conservan como parte de la practica entregada. Antes de llevar este reto a la web de Chilete DevPath, deben reemplazarse por assets propios o recursos con fuente/licencia clara.

## Reto 2: La Cafeteria Asincrona

Practica de JavaScript asincrono.

Tecnologias:

- JavaScript
- Promesas
- `async/await`
- `setTimeout`
- Manejo de errores

Ejecucion:

```powershell
cd reto2-cafeteria-asincrona
node app.js
```
