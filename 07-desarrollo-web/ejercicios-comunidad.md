# Ejercicios para la comunidad

Estos retos complementan los ejercicios existentes de `07-desarrollo-web`.

## HTML

1. Crear una pagina de presentacion con encabezado, secciones y enlaces.
2. Organizar una lista de recursos de aprendizaje.
3. Crear una tabla simple con temas, estado y observaciones.

## CSS

1. Estilizar una tarjeta de recurso.
2. Crear una barra de navegacion responsive.
3. Usar Flexbox para ordenar tarjetas en escritorio y movil.

## JavaScript

1. Mostrar u ocultar una seccion al hacer clic.
2. Validar un formulario antes de enviarlo.
3. Simular una carga asincrona de recursos.

## PHP

1. Procesar un formulario con datos ficticios.
2. Calcular el promedio de notas enviadas desde un formulario.
3. Mostrar una lista generada desde un arreglo.
4. Registrar datos en una practica local sin usar informacion real.

## Criterio para resolver

Antes de publicar una practica:

1. Revisa que no tenga datos personales reales.
2. Comprueba que funcione en navegador.
3. Explica que archivo contiene estructura, estilos, interaccion o procesamiento.
4. Usa textos propios de Chilete DevPath cuando sea natural.
5. Evita imagenes o marcas externas sin permiso.
