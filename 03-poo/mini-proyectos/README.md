# Mini Proyectos

Ejercicios integradores de programacion orientada a objetos.

## Contenido

- `java/comprobantes`: validacion simple de cliente y comprobante.
- `java/persona-serializacion`: practica de serializacion con datos ficticios.
- `java/practica-integrada-poo`: ejercicios con listas y objetos.
- `java/tres-en-raya`: juego simple para practicar clases y responsabilidades.

## Criterio

Estos ejercicios combinan varios conceptos de POO. Si se llevan a la web, deben presentarse como casos propios de Chilete DevPath y con datos ficticios.
