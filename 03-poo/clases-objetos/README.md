# Clases y Objetos

Ejercicios sobre creacion de clases, atributos, constructores, metodos e instancias en Java.

## Contenido

- `java/caja-generica`: uso inicial de clases genericas.
- `java/enumeraciones-aplicadas`: uso de `enum` para representar valores fijos.
- `java/libro-basico`: clase simple con atributos y comportamiento.
- `java/objetos-basicos`: practica de multiples clases con datos ficticios.
- `java/perfil-estudiante`: modelado basico de persona/estudiante.
- `java/plantilla-vehiculo-generica`: plantilla simple aplicada a vehiculos.
- `java/rectangulo-basico`: clase con calculo de area.
- `java/registro-estudiantes-generico`: registro usando tipos genericos.

## Criterio

Los datos usados son ficticios y sirven para practicar modelado. Si se agregan nuevos ejemplos, deben evitar datos personales reales.
