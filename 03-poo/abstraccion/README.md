# Abstraccion

Ejercicios sobre clases abstractas, interfaces y modelado de comportamientos comunes.

## Contenido

- `java/figuras-abstractas`: figuras con comportamiento comun y calculo especializado.
- `java/interfaces-y-pagos`: interfaces para separar comportamiento de implementacion.
- `java/notificaciones-abstractas`: notificaciones modeladas con abstraccion.

## Criterio

La abstraccion se usa cuando varias clases comparten una idea comun, pero cada una necesita implementar su propio comportamiento.
