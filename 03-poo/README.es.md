# Programacion Orientada a Objetos

[English version](README.md)

Esta carpeta contiene ejercicios, apuntes y ejemplos relacionados con programacion orientada a objetos.

## Temas

- `classes-and-objects`: modelado de entidades con clases y objetos.
- `encapsulation`: control de acceso y proteccion de datos.
- `inheritance`: jerarquias de clases y reutilizacion de codigo.
- `polymorphism`: comportamiento flexible con sobrescritura e interfaces.
- `abstraction`: clases abstractas, interfaces y modelado esencial.
- `uml`: diagramas de clases y modelado visual de sistemas.
- `mini-projects`: pequenos proyectos que combinan conceptos de POO.
